# Start Project template.


## Requisitos

- Python 3.12
- Poetry

## Como usar

### 1. Instale as dependências

```bash
poetry install
```

### 2. Rodar os comandos de formatação e linting

```bash
poetry run task format
```

### 3. Rodar os testes

```bash
poetry run pytest
```

## Estrutura do Projeto

- `src/`: Código fonte do projeto
- `tests/`: Testes automatizados
